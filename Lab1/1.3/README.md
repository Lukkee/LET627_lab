# Step 3: Deadlines to the rescue
## Controls
### Volume
***m*** - mute / unmute
***+*** - volume up (max 20)
***-*** - volume down (min 0)

### Background Load and Deadline
***j*** - toggle deadlines
***u*** - increase background load
***d*** - decrease background load

### Frequencies
***1*** - 1 kHz
***2*** - 769 Hz
***3*** - 537 Hz
